Mainumby es una aplicación experimental que ayuda a traducir del castellano al guaraní. Se está desarrollando dentro del proyecto Procesamiento de Idiomas del Sur Global (PISUR), dirigido por Prof. Michael Gasser.

  • Para usarlo para la traducción de una palabra, una frase, o una oración simple, seleccioná ORACIÓN bajo TRADUCIR.
  • Para la traducción de un documento, seleccioná DOCUMENTO bajo TRADUCIR.
  • Para identificarte antes de usar el traductor, seleccioná IDENTIFICARTE.
